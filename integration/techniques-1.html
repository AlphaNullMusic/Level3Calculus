Integration

- Integrating Polynomials

Differentiation is the process of finding the gradient of a curve at any point on that curve. Integration is the reverse process to differentiation.


For Example:

If \(f(x) = 3x^2 + 2x^2 + 6\) then \(f'(x) = 9x^2 + 4x\) (differentiation).

If \(f'(x) = 9x^2 + 4x\) then \(f(x) = 3x^2 + 2x^2 + ?\) (integration).


The general rule to follow when integrating is: \[\int{ax^n dx} = \frac{a}{n+1}x^{n+1} + c\]

The \(c\) at the end represents the constant of integration. This is an unknown number as a number with no variables differentiates to zero. The constant could be any number and still differentiate to zero. However, we still need to allow for the possibility of this constant being present. Thus, we have the constant of integration (\(c\)) to account for this.


Note: when integrating \(c\), it becomes \(cx\).

For Example:

• \(8x^3 - 2x^2 + 5dx\).

This question tells us to integrate the function \(f(x) = 8x^3 - 2x^2 + 5\) with respect to \(x\).


Integrating will give us: \[\frac{8}{4}x^4 - \frac{2}{3}x^3 + 5x + c\]

We can simplify fractions further to get the final answer: \[2x^4 - \frac{2x^3}{3} + 5x + c\]

- Integrating Rational Functions and Surds

Similarly to differentiation of rational functions, there are six situations for integration that you need to know:

  1. If the function is already in index form, then use the rule directly: \[f(x) = 8x^3 - 2x^2 + 5\] \[\int{f(x)dx = 2x^4 - \frac{2x^3}{3} + 5x + c}\]
  2. If the function has fractions and x is on the numerator, seperate the fraction and use the rule: \[\eqalign{f(x) &= \frac{3x^2}{4} + 2x \\ &= \frac{3}{4}x^2 + 2x}\] \[\int{f(x)dx = \frac{3}{12}x^3 + x^2 + c}\]
  3. If the function has fractions and x is on the denominator, rewrite the x term with a negative index, use the rule, then convert the answer back to the form the question was in: \[\eqalign{f(x) &= \frac{5}{4x^3} + \frac{x}{2} \\ &= \frac{5}{4}x^{-3} + \frac{1}{2}x}\] \[\eqalign{\int{f(x)dx} &= -\frac{5}{8}x^{-2} + \frac{1}{4}x^2 + c \\ &= -\frac{5}{8x^2} + \frac{x^2}{4} + c}\]
  4. If the function is a product of factors, expand out the brackets: \[\eqalign{f(x) &= (x+2)(x-4) \\ &= x^2 - 2x - 8}\] \[\int{f(x)dx = \frac{x^3}{3} - x^2 - 8x + c}\]
  5. If the function is expressed as a complete fraction, with x on both sides, break the function up, simplify each one and use the rule: \[\eqalign{f(x) &= \frac{5x^4 + 3x^2 + 2x}{2x} \\ &= \frac{5x^3}{2} + \frac{3x}{2} + 1}\] \[\int{f(x)dx = \frac{5x^4 + 6x^2 + 8x}{8} + c}\]
  6. If the fraction has surds, convert them to indicies (remembering that \(\sqrt[n]{x^m} = x^{\frac{m}{n}}\)), and use the rule: \[\eqalign{f(x) &= \sqrt[3]{x} + \sqrt[4]{x^2} \\ &= x^{\frac{1}{3}} + x^{\frac{2}{4}}}\] \[\int{f(x)dx = \frac{3\sqrt[3]{x^4}}{4} + \frac{2\sqrt{x^3}}{3} + c}\]

- Finding the Constant of Integration

In the exam, there are two typical contexts which require you to find the constant of integration:

  • - Kinematics
  • - Gradients and curves

For example, if we were told that the derivative of \(f(x)\) is 5, we cna find that \(f(x) = 5x + c\), but to calculate \(c\) we need more information. In this example, we know the gradient of the graph, but not the y-intercept value. If we knew a co-ordinate that was on the curve of \f(x)\), then we could find the constant of integration with algebra.

Example:

• Find the function \(f(x)\) which does through the point (1, 10) and has a derivative \(f'(x) = 5\).

We know: \[\int{5dx = 5x + c}\] \[f(x) = 5x + c\]

We also know that the function goes through the co-ordinate (1, 10), so: \(10 = (5 * 1) + c\). Rearranging this gives us \(c = 5\).

Therefore the full function is \(f(x) = 5x + 5\).

- Integrating Exponentials

We know that if you differentiate \(f(x) = e^x\) then you get \(f'(x) = e^x\). Therefore, to integrate an exponential function it follows that: \[\int{e^x dx} = e^x + c\]

If the function has coefficients, such as in \(f(x) = ae^{kx}\) then \(f'(x) = (a * k)e^{kx}\). This means that the general rule for integrating exponential functions is: \[\int{ae^{kx}dx} = \frac{ae^{kx}}{k} + c\]


Examples:

  1. \[\eqalign{\int{8e^{3x} dx} &= \int{(\frac{8}{3} * 3)e^{3x} dx} \\ &= \frac{8e^{3x}}{3} + c}\]

  2. \[\eqalign{\int{\sqrt{e^{2x}} dx} &= \int{e^{\frac{2x}{2}} dx} \\ &= \int{e^x dx} \\ &= e^x + c}\]

    3. - Integrating Trigonometric Functions

    All basic trigonometric functions can be integrated to a different one. You will be given a table with the original and integral functions for your exam.


    Function (\(y= f(x)\)) Derivative (\(\int{f(x)dx + c}\))
    \(\cos{x}\) \(\sin{x} + c\)
    \(\sin{x}\) \(-\cos{x} + c\)
    \(\sec^2{x}\) \(\tan{x} + c\)
    \(\csc{x} \cot{x}\) \(-\csc{x} + c\)
    \(\sec{x} \tan{x}\) \(\sec{x} + c\)
    \(\csc^2{x}\) \(-\cot{x} + c\)

    Note:

    - \(\csc{x}\) can also be expressed as \(\text{cosec } x\).

    - trigonometric functions are often written using this notation: \((\sin{x})^2\) is written as \(\sin^2{x}\).

    - use the reverse chain rule for composite functions (e.g: \(\int{\sin{(g(x))} dx}\).

    - you may be given a coordinate on the original curve, in which case you can determine the constant of integration using the same method as with polynomials.

    Example:

    • \(\int{5\cos{(2x + 1)} dx}\).

    From the table, the integral of \(\cos{x}\) is \(\sin{x}\).

    The derivative of \((2x + 1\) is 2, and we need to multiply 2 by \(\frac{5}{2}\) to get 5. Therefore, the answer is: \[\int{5\cos{(2x + 1)} dx} = \frac{5}{2}\sin{(2x+1)} + c\]

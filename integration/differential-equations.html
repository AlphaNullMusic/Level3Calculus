Integration

- Particular Solutions

A differential equation is an equation that includes a derivative, such as: \(\frac{dy}{dx}, \frac{d^2 y}{dx^2}, f'(x), f''(x), y', y''\), etc. To solve them, we need to integrate.

For example, to solve the differential equation \(y' = 8x + 3\), we would integrate to get \(y = 4x^2 + 3x + C\). This answer is called the general solution to the equation, because we haven't identified the value of the constant of integration.

Example:

• Find the general solution to \(f'(x) = 15x^3 + 12x^2 + 3x - 2\).


The answer is: \[\eqalign{f(x) &= \int{(15x^3 + 12x^2 + 3x - 2)}\\ &= \frac{15x^4}{4} + 4x^3 + \frac{3x^2}{2} + 2x + C}\]


To find the particular solution of a differential equation, we must integrate it and then find the value of the constant of integration (\(c\)).

Example:

• Consider the differential equation \(y' = 8x + 3\), which passes through the point (1, 8).


Because we know a point which it passes through, we can integrate and then solve to find \(c\): \[y = 4x^2 + 3x + C\] \[\eqalign{8 &= 4(1)^2 + 3(1) + C \\ &= 4 + 3 + C}\] \[C = 1\]

Therefore the particular solution to this equation is \(y = 4x^2 + 3x + 1\).

- Second-Order Differential Equations

The differential equations we have covered so far only contain a first derivative. They are called First Order Differential Equations.


We also need to know how to solve Second Order Differential Equations. To do so, we integrate the equation twice, remembering to include a different constant of integration for each time we integrate. These can be labelled so we can tell them apart. For example: \(C_{1}\) first and then \(C_{2}\).


Example:

• Solve the differential equation \(\frac{d^2 y}{dx^2} = 20x^3 + 6x\), given that when \(x = 3\), \(y = 5\) and \(\frac{dy}{dx} = 10\).


First we integrate to get a first order differential equation: \[\eqalign{\frac{dy}{dx} &= \int{(20x^3 + 6x) dx}\\ &= 5x^4 + 3x^2 + C}\]

We know that \(\frac{dy}{dx} = 10\) when \(x = 3\), so we can calculate the constant of integration: \[\eqalign{10 &= 5(3)^4 + 3(3)^2 + C_{1} \\ &= 81 + 27 + C_{1}}\] \[C_{1} = -91\] \[\frac{dy}{dx} = 5x^4 + 3x^2 - 98\]

Now we integrate again to get the general solution: \[\eqalign{y &= \int{(5x^4 + 3x^2 - 98) dx} \\ &= x^5 + x^3 - 98x + C_{2}}\]

Finally, we know that when \(x = 3\), \(y = 5\), so we can find the particular solution: \[\eqalign{5 &= 3^5 + 3^3 - (98 * 3) + C_{2} \\ &= 243 + 27 - 294 = C_{2}}\] \[C_{2} = 29\]

Therefore \(y = x^5 + x^3 - 98x + 29\).

- Seperated Variables

Sometimes we may get a differential equation with two seperate part, such as: \[\frac{dy}{dx} = f(x) * g(y)\] In this instant \(f(x)\) is a function of \(x\), but \(g(y)\) is a function that only involves \(y\).

To solve this problem, we must write the derivative as either a product or quotient of two functions.


An example of this form is the equation: \[\frac{dy}{dx} = (2 + x^2)(2y - 3)\]

The best way to solve equations in this form is to re-write it so that everything in terms of \(y\) is one one side and everything in terms of \(x\) is on the other side.


Note: because the derivative can be treated as a ratio, always use Leibniz notation (\(\frac{dy}{dx}\)) for this.

If we start with the general form: \[\frac{dy}{dx} = f(x) * g(y)\]

We can multiply both sides by \(dx\) and divide both sides by \(g(y)\) to give: \[\frac{1}{g(y)} dy = f(x) dx\]

Integrating both sides will give us: \[\int{\frac{1}{g(y)} dy} = \int{f(x) dx}\]

The left hand side is now integrated with respect to \(y\) and the right hand side with respect to \(x\).

The solution will therefore be expressed in the form: \(G(y) = F(x)\), where \(G(y)\) is the integral of \(g(y)\) and \(F(x)\) is the integral of \(f(x)\).

Example:

• Find the general solution of the differential equation: \(\frac{dy}{dx} = \frac{x}{15y^2}\).


First we multiply both by \(dx\) and \(15y^2\): \[15y^2 dy = x dx\]

Then we integrate them to get the general solution: \[\int{15y^2 dy} = \int{x dx}\] \[5y^3 = \frac{x^2}{2} + C\]

Note: it is a good idea to check your answer. To do so, you can substitute the differential of your answer back into the original question, and both sides should cancel out.

- Applications of Differential Equations

Differential equations are very useful when applied in fields like medicine and finance.

An equation with the form \(\frac{dy}{dx} = ky\) can be used to solve problems ranging from depreciation, population growth, temperature change, and radioactive decay.


To see why it's so useful, lets solve it: \[\frac{dy}{dx} = ky\] \[(\frac{1}{y})dy = (k)dx\] \[\int{(\frac{1}{y}) dy} = \int{(k) dx}\] \[\ln{(y)} = kx + C\] \[\eqalign{y &= e^{kc + C} \\ &= e^{kx}e^{C}}\]

Therefore: \(y = y_{0}e^{kx}\) where \(y_{0} = e^C\).

You may be able to see know that this is the general formula for exponential growth/decay, as you may have used in your exams last year.

Example:

• Mark has a large stamp collection. If the rate at which the value of the stamp appreciate depends on the value of the stamps, find the annual rate of appreciation for the collection. Mark purchased the collection at the beginning of 2012 for $2300 and it was valued at $3400 at the end of 2015.


The general solution is: \[P = P_{0}e^{it}\] \[3400 = 2300e^{4i}\] \[\eqalign{e^{4i} &= \frac{3400}{2300} \\ &= 1.478261}\] \[4i = \ln{(1.478261)}\] \[\eqalign{i &= \frac{\ln{(1.478261)}}{4} \\ &= 0.09772 \\ &= 9.8\text{%}}\]

- Newton's Law of Cooling

In the exam, you may be given a question about Newton's Law of Cooling. This is the formula which models how the surface temperate of an object changes at a rate proportional to the difference between its temperature and the temperature of the surrounding environment.


It is modelled using a differential equation: \[\frac{dT}{dt} = r(T - T_{s})\]

where:

  • \(T\) is the temperature of the object.
  • \(t\) is the time taken.
  • \(r\) is the rate of change of temperature per time period.
  • \(T_{s}\) is the temperature of the surrounding area.

We may need to find a general solution to this equation to answer a question in context: \[\frac{1}{T - T_{s}}dT = (r)dt\] \[\int{(\frac{1}{T - T_{s}}) dT} = \int{(r) dt}\] \[\ln{(T - T_{s})} = rt + C\] \[\eqalign{T - T_{s} &= e^{rt + C} \\ &= ke^{rt}}\] \[T = T_{s} = ke^{rt}\] where \(k = e^C\).


Example:

• A pizzeria has a contant temperature of 18°C. The rate at which the pizza oven tray changes at any instant is proportional to the difference between the temperature of the oven tray and the room temperature at that instant.


The temperature of the oven tray is 260°C when it gets taken out. After 3 minutes its temperature is 100°C. Solve the differential equation \(\frac{dT}{dt} = r(T - 18)\) to find what temperature it will be after another 4 minutes.


Rearranging and solving for \(C\): \[\frac{dT}{dt} = r(T - 18)\] \[T = T_{s} + Ce^{rt}\] \[\eqalign{260 &= 18 + Ce^{0 * r}\\ &= 18 + Ce^0 \\ &= 18 + C}\] \[C = 242\]

Now we can solve for \(r\) by substituting \(t = 3\), \(C = 242\), and \(T = 100\): \[100 = 18 + 242e^{3r}\] \[82 = 242e^{3r}\] \[\frac{82}{242} = e^{3r}\] \[ln(0.3388) = 3r\] \[r = -0.3607\] \[T = 18 + 242e^{-0.3607t}\]

Finally, we solve for 4 minutes later, which means 7 minutes in total: \[\eqalign{T &= 18 + 242e^{-0.3607 * 7}\\ &= 64.8^{\circ}\text{C}}\]

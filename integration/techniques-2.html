Integration

- Integrating Products (Reverse Chain Rule)

The chain rule says that differentiating a composite function gives a product or two functions: \[\frac{d}{dx}f(g(x)) = f'(g(x)) * g'(x)\]

To integrate a composite function, we can use this rule in reverse, so: \[\int{f'(g(x)) * g'(x)dx = f(g(x)) + c}\]


Example:

• \(\int{3(x^2 + 3)^2 * 2xdx}\).

Use the reverse chain rule: \[\int{3(x^2 + 3)^2 * 2xdx} = (x^2 + 3)^3 + c\]

- Integration of Inverses

We know from differentiation that the derivative of \(\ln{x}\) is \(\frac{1}{x}\), so it makes sense that: \[\int{\frac{1}{x} dx} = \ln{|x|} + c\]

Note: you can't take the log of a negative number, so we have to use the absolute value of \(x\) (\(|x|\)). The absolute value is always positive. If \(x\) is negative, the absolute value sign effectively removes the negative sign, making it positive.

We can simplify this expression further by defining a new variable: \(k = e^c\). Rearranging this will give us \(c = \ln{(k)}\) and we can use the logarithm laws to simplify the formula: \[\eqalign{\int{\frac{1}{x} dx} &= \ln{|x|} + c \\ &= \ln{|x|} + \ln{k} \\ &= \ln{|kx|}}\]


Example:

• Find the integral of the following expression: \[\int{\frac{8dx}{8x - 2}}\]

We can rewrite the expression like so: \[\eqalign{\int{\frac{8dx}{8x - 2}} &= \int{\frac{8}{8x - 2} dx} \\ &= 8 \int{\frac{1}{8x - 2} dx}}\]

Now we can use the formula to find the integral: \[8 * \frac{1}{8}\ln{|8x - 2|} + c = \ln{|8x + 2| + c}\]

- Integrating Quotients

There are three different situations for integrating quotients:


  1. Quotients of the Form \((ax + b) / x\)


    If we get an expression of the form \(\frac{ax + b}{c}\), we can integrate by splitting it up into parts: \[\frac{ax + b}{c} = \frac{ax}{c} + \frac{b}{c}\]

    Note: you can't split up the denominator like so: \[\frac{c}{ax + b} \ne \frac{c}{ax} + \frac{c}{b}\]

  2. Quotients of the Form \((ax + b) / (cx + d)\)


    For expressions of this type, look to factorise where possible and then 'break up' the fraction like the previous

    Example:

    \[\eqalign{\int{\frac{12x + 8}{3x + 4} dx} &= \int{\frac{4(3x + 4) - 8}{3x + 4} dx}\\ &= \int{4 - \frac{8}{3x+4} dx} \\ &= 4x - \frac{8\ln{|3x + 4|}}{3} + c}\]

  3. Quotients of the Form \(f'(x)/f(x)\)


    From differentiation, we know that if \(y = \ln{x}\) then \(\frac{dy}{dx} = \frac{1}{x}\).

    We also know that if \(f(x) = \ln{(g(x))}\) then \(f'(x) = \frac{g'(x)}{g(x)}\).

    From this we can find that: \[\int{\frac{f'(x)}{f(x)} dx} = \ln{|x|} + c\] If you see a fraction where the numerator is the direct derivative of the denominator, integrate directly using the form above.

    Example:

    \[\int{\frac{6e^{2x} + \cos{x}}{6e^{2x} + 2\sin{x}} dx}\]

    The derivative of \(6e^{2x} + 2\sin{x}\) is \(12e^{2x} + 2\cos{x}\). If we multiply it by \(\frac{1}{2}\) then we get the numberator. Therefore, the answer is: \[\frac{1}{2} \ln{|6e^{2x} + 2\sin{x}|} + c\]

- Integration by Substitution

When the function needing the be integrated is expressed as a composite function, you will be required to integrate it by substitution.


Remember that a composite function is a function within a function.

If we had two functions: \(f(x) = x^5\) and \(g(x) = x^3 - 2x + 1\), then \(f(g(x))\) would mean that \(g(x)\) was substituted in place for \(x\) in \(f(x)\). The resulting composite function is: \(f(g(x)) = (x^3 - 2x + 1)^5\).


The process to integrate a function like this is to declare variables, like \(u = x^3 - 2x + 1\) and then substitute them at the end. The process can be seen in the example below:


Example:

• Integrate \(6x(x^2 + 10)^5 dx\).


First, we let \(u = x^2 + 10\), which means \(\frac{du}{dx} = 2x\).

Then we multiply both sides by \(dx\) to give \(du = 2x dx\).


In the original function we have \(6x\) at the start and \(dx\) at the end.

So, if \(du = 2x dx\) then \(3du = 6x dx\).


We can now substitute \(3du\) into the function to replace \(6x dx\), which makes it easier to integrate: \[\int{3(u)^5 du} = 12u^6 + c\]

Finally we substitute \(u\) back into to give the completed integral: \(\frac{1}{2}(x^2 + 10)^6 + c\)

Note: you will be required to identify an appropriate substitution before you can use it to integrate in the exam.

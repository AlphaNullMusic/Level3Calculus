Integration

- The Rectangle Rule

Numerical integration is a way of estimating the definite integral. The simplest of the three methods needed for your exam is the Rectangle Rule.


This method divides the area underneath the curve into a series of rectangles, which are then summed up to give an approximate total area underneath the curve.

To do this, we need to know the boundary values and the number of rectangles to make (sometimes referred to as the number of sub-intervals).


To find the area of each rectangle, we have to find the height of each rectange, which is done by working out the mid-point of each (we already know the width). We can then substitute the mid-point of each rectangle into the function and generate a table of values.


Example:

• Find an approximate area underneath the function \(y = 6\log{(x)} + 2\) between \(x = 2\) and \(x = 4\) using the rectangle rule with four subintervals.


Firstly, it is good to sketch the graph and rectangles to make sure your answer is reasonable accurate.

Then we find the width of the rectangles by dividing the difference of the range by the number of rectangles: \(\text{width (w)} = \frac{4 - 2}{4} = 0.5\).


Finally, we can generate a list with values for each rectangle:

  • - \(\text{midpoint }(x_{n}) = 1.75 + wn\)

    • Rectangle 1 = 2.25
    • Rectangle 2 = 2.75
    • Rectangle 3 = 3.25
    • Rectangle 4 = 3.75

  • - \(\text{height }(y_{n}) = f(x_{n})\)

    • Rectangle 1 = 4.113
    • Rectangle 2 = 4.636
    • Rectangle 3 = 5.071
    • Rectangle 4 = 5.444

  • - \(\text{area}(a) = w * y_{n}\)

    • Rectangle 1 = 2.0565
    • Rectangle 2 = 2.318
    • Rectangle 3 = 2.5355
    • Rectangle 4 = 2.722

  • - the total area is 9.632

- The Trapezium Rule

Another, more accurate, way of approximating an indefinite integral is to use a series of trapeziums. It will produce the best estimation out of all three methods.

If we do the same thing as with the rectangle rule, and have trapezia of the same width (\(h\)), we can create a general formula. The area of one trapezium is given by: \[\eqalign{\text{area} &= \frac{a+b}{2} + h \\ &= \frac{h}{2}(a + b)}\]

We can generalise this to give (remembering that there is one more \(y\) value than the number of trapezia: \[T = \frac{h}{2}(y_{0} + 2y_{1} + 2y_{2} + ... + 2y{n-1} + y_{n})\]

Example:

• Use the trapezium rule to find the area underneath \(y = -x^2 + 4\) between \(x = 0\) and \(x = 2\) using four subintervals.


For this example, \(h = \frac{2}{4} = 0.5\).

You will have to generate a table of values, as with the rectangle rule:

\(n\) \(x_{n}\) \(y_{n}\)
0 0 4
1 0.5 3.75
2 1 3
3 1.5 1.75
4 2 0

Finally, you will substitute the values into the trapezium rule: \[\eqalign{T &= \frac{h}{2}(y_{0} + 2y_{1} + 2y_{2} + ... + 2y{n-1} + y_{n}) \\ &= \frac{0.5}{2}(4 + 2 * 3.75 + 2 * 3 + 2 * 1.75 + 0) \\ &= \frac{0.5}{2}(4 + (2 * 8.5)) \\ &= 5.25}\]

Integrating \(\int_{0}^{2}{(-x^2 + 4) dx}\) would give us \(5\frac{1}{3}\), which is pretty close to our approximation of 5.25.

- Simpson's Rule

Finally, we have Simpson's Rule, which uses parabolas to approximate the area under a curve: \[\int_{a}^{b}{f(x) dx} \approx \\ \frac{h}{3}(y_{0} + y_{n} + 4(y_{1} + y_{3} + ... + y_{n-1}) + \\ 2(y_{2} + y_{4} + ... + y_{n-2}))\]

In this formula, \(h\) represents the width of the range that each parabola approximates.

Note: the Simpson's Rule will only work with an even number of sub-intervals.

Example:

• Use Simpson's Rule to find the area underneath \(y = -x^2 + 4\) between \(x = 0\) and \(x = 2\) using four subintervals.


Get the value of \(h\): \[\eqalign{h &= \frac{2 - 0}{4} \\ &= 0.5}\]

Now we generate a table of values:

\(y\) number \(x\) value \(y\) value
\(y_{0}\) 0 4
\(y_{1}\) 0.5 3.75
\(y_{2}\) 1 3
\(y_{3}\) 1.5 1.75
\(y_{4}\) 2 0

Then we substitute these into the formula: \[\eqalign{\int_{a}^{b}{f(x) dx} &\approx \frac{h}{3}(y_{0} + y_{4} + 4(y_{1} + y_{3}) + 2(y_{2})) \\ &\approx \frac{0.5}{3}(4 + 0 + 4(3.75 + 1.75) + 2(3))\\ &\approx \frac{1}{6}(4 + 22 + 6) \\ &\approx 5\frac{1}{3}}\]

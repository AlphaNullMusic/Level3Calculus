Integration

- Definite Integrals

A definite integral is a special type of integral where you're asked to find the value of the integral between two points. It is written in this notation: \[\int_{a}^{b}{f(x) dx}\]

This expression means that you take the value of the integral at \(x = b\) and subtract the value at \(x=a\) from it.

We can define the integrated function \(\int{f(x)} = F(x)\), so it becomes: \[\int_{a}^{b}{f(x) dx} = (F(b) + c) - (F(a) + c)\]


Example:

• Evaluate \(\int_{2}^{4}{2x^3 + 5 dx}\)


First we find the general integral: \[\int{2x^3 + 5 dx} = \frac{1}{2}x^4 + 5x + c\]

We can then evaluate this at \(x = 2\) and \(x = 4\) to find the answer. There is a special notation to denote substituting values into an integrated equation: \[[\frac{1}{2} x^4 + 5x]_{2}^{4}\]

By substituting our \(x\) values we get the answer: \[\eqalign{[\frac{1}{2} x^4 + 5x]_{2}^{4} &= (\frac{1}{2} * 4^4 + 5 * 4) - (\frac{1}{2} * 2^4 + 5 * 2) \\ &= 148 - 18 \\ &= 130}\]

If we know the rate at which a quantity \(f(x)\) is changing (the rate of change), then we can use definite integrals to calculate the total change in \(f(x)\) over a fixed period: \[\int_{1}^{2}{f'(x) dx} = f(2) - f(1)\] This equation will tell us how much \(f(x)\) has changed between \(x = 1\) and \(x = 2\).

- Kinematics

A very common context for integral problems in the exam is kinematics, which is the movement of object in the physical world.


A very important thing to remember is that is a function gives the displacement (\(s\)) of an object, the first derivative gives the velocity (\(v\)) of the object, and the second derivative will represent the acceleration (\(a\)) of the object at a specific time.

Therefore, we can say that \(f(t) = s\), \(f'(t) = v\) and \(f''(t) = a\).

Remember these fundamentals for kinematics:


- replacing time with zero will give the initial value of displacement, velocity or acceleration.

- solving the equation when it equals zero will give the time when the object is at the starting point, at rest, or not accelerating.

- is \(v > 0\) then the object is moving away from the start position, and if \(v < 0\) it is moving back towards the start.

- if \(a > 0\) then the object is speeding up, if \(a = 0\) then the object is at a constant velocity, and if \(a < 0\) then the object is slowing down.

Example:

Troy is flying a toy helicopter. It begins on the picnic table, flies away from Troy for some distance and then flies back to him. The equation that represents the helicopter's distance from the picnic table after \(t\) seconds is given by: \(s = -0.5t^2 + 4t\). Distance is measured in metres and time in seconds.


  • i) How far has the toy helocipter travelled from the picnic table after 5 seconds?

    To find displacement, we substitute \(t = 5\) into the function: \[\eqalign{s &= -0.5t^2 + 4t \\ &= -0.5 * 5^2 + (4 * 5) \\ &= -0.5 * 25 + 20\\ &= 7.5m}\]

  • ii) At what time does the helicopter stop moving away and start moving back towards Troy?

    This situation will occur when \(s = 0\) and is sometimes referred to as the object's fixed point.

    Solving \(0 = -0.5t^2 + 4t\) gives \(t = 0\) and \(t = 4\) seconds. The first answer is when the helicopter starts flying from the table, so the answer is at 4 seconds.

  • iii) What is the function that represents the velocity of the helicopter?

    Remember that \(v = s' = \frac{ds}{dt}\)

    So, differentiating the original function will give us the function of velocity: \(v = -t + 4\text{m/s}\).

  • iv) What was the helicopter's initial velocity?

    Initial velocity happens when \(t = 0\), so the answer is: \[\eqalign{v &= -(0) + 4 \\ &= 4\text{m/s}}\]

  • v) What is the acceleration of the helicopter?

    Remember that \(a = v' = s'' = \frac{dv}{dt}\)

    So, differentiating \(v\) gives us: \(a = -1\text{m/s}^2\).

You will have to use a mixture of differentiation and integration to solve kinematic equations, so make sure that you're comfortable with differentiating and integrating these equations, as well as reading word problems.

- Area Under a Curve

One of the reasons integration was invented was to find the inverse transformation of differentiation (the anti-derivate). The other reason was to find the area of a region.


If you're given two endpoints of the function, you can find the area the function covers by simply integrating it with a definite integral. If the graph is above the x-axis, this will work fine. If the curve is completely or partially below the x-axis, we need to take the absolute value of the area (\(|a|\)), as area can't be negative.

- Area Above and Below the x-Axis

For curves which are partially above and below the x-axis, taking the integral of the function will be useless, as the negative parts will cancel out.


Instead, we have to split the integrals into above and below, and make sure to take the absolute value of the area below the x-axis.


Example:

• Find the area underneath the first period (\(2\pi\)) of the curve \(f(x) = sin(x)\).


To get the answer, we need to split the integrals and the substitute like so: \[\eqalign{\text{Area} &= \int_{0}^{\pi}{\sin{(x)} dx} + |\int_{\pi}^{2\pi}{\sin{(x)} dx}| \\ &= (-\cos{\pi} + \cos{0}) + (|-\cos{2\pi} + \cos{\pi}|) \\ &= 2 + |-2| \\ &= 2 + 2 \\ &= 4}\]

- Area Between Two Curves

The next step is to find the difference (area) between two curves. To do this, you can rewrite the problem as the difference between two functions: \[\text{Area} = \int_{a}^{b}{f(x) dx} - \int_{a}^{b}{g(x) dx}\]


Example:

• Find the difference between \(\cos{(x)}\) and \(\sin{(x)}\) between 0 and the first point of intersection.


Solving \(\sin{(x)} = \cos{(x)}\) gives us a value of \(x = \frac{\pi}{4}\).

This gives us boundaries of \(x = 0\) and \(x = \frac{\pi}{4}\).

Now we take the integral of each function: \[\eqalign{\int_{0}^{\frac{\pi}{4}}{\cos{(x)} dx} &= (\cos{(\frac{\pi}{4})} - \cos{(0)}) \\ &= 0.7071 \text{ (4 d.p.)}}\] \[\eqalign{\int_{0}^{\frac{\pi}{4}}{\sin{(x)} dx} &= (\sin{(\frac{\pi}{4})} - \sin{(0)}) \\ &= 0.2928 \text{ (4 d.p.)}}\]

Subtracting the integral of the \(sin\) curve from the integral of the \(cos\) curve will give us the answer: \(0.7071 - 0.2928 = 0.4143\).

Note: in some questions you may be given the area and have to figure out the range of the integral, instead of the other way around. This can be done by substituting \(x\) as the unknown value in the range and then solve by integrating and solving for \(x\).

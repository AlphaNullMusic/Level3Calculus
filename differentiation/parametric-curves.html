Differentiation

- Parametric Equations

A parametric equation has variables defined in terms of another single parameter, often the letter \(t\).

We can use the chain rule to differentiate parametric equations with the chain rule: \[\frac{dy}{dx} = \frac{dy}{dt} * \frac{dt}{dx}\]


Example:

• Differentiate the pair of parametric equations: \(x = 3t\) and \(y = 6t - 1\).

Differentiating each one with respect to \(t\) gives us: \[\frac{dx}{dt} = 3\] and \[\frac{dy}{dt} = 6\]

We can now substitute this into the chain rule to give the answer: \[\eqalign{\frac{dy}{dx} &= \frac{dy}{dt} * \frac{dt}{dx} \\ &= 6 * \frac{1}{3} \\ &= 2}\]


Note: to find the second derivative of a parametric equation, use the formula: \[\frac{d^2 y}{dx^2} = \frac{d(\frac{dy}{dx})}{dt} * \frac{dt}{dx}\]

Proof:

• Find the second derivative given the parametric equations: \(x = 3t^3\) and \(y = 8t\).

We know that \(frac{dy}{dt} = 8\), \(frac{dx}{dt} = 9t^2\) and \(frac{dt}{dx} = \frac{1}{9t^2}\).

Now, using the chain rule, we get: \[\eqalign{\frac{dy}{dx} &= \frac{dy}{dt} * \frac{dt}{dx} \\ &= 8 * \frac{1}{9t^2}}\]

We need to differentiate this again, so we can express the second derivative as: \[\frac{d^2 y}{dx^2} = \frac{d}{dx}(\frac{dy}{dx})\]

Then we rearrange it and apply the chain rule: \[\eqalign{\frac{d(\frac{dy}{dx})}{dx} &= \frac{d(\frac{dy}{dx})}{dt} * \frac{dt}{dx} \\ &= \frac{d}{dt}(\frac{8}{9t^2}) * \frac{1}{9t^2} \\ &= \frac{-16}{9t^3} * \frac{1}{9t^2} \\ &= \frac{-16}{81t^5}}\]

You may be asked to sketch a parametric curve and draw the tangent to the curve at a point. To do this, you should draw a table of values by substituting integers for \(t\) into \(x\) and \(y\).

Use this table to draw the graph and then draw a tangent at the specified point. You can then use \(y = mx + c\) to find the equation for the tangent, or the straight-line formula: \((y - y_{1}) = m(x - x_{1})\), where \((x_{1},y_{1})\) is the given co-ordinate on the straight line.


You might also be asked to draw the normal, which is perpendicular to the tangent. The equation is also give by the stright-line formula. The gradient of the normal \(m_{N}\) is the negative reciprocal of the gradient of the tangent \(m_{t}\).

This means that if \(m_{t} = 8\) then \(m_{N} = -\frac{1}{8}\).

- Implicit Differentiation

A normal function is expressed explicitly. This means that one variable is expressed in terms of another. For example: \(y = x^2 + 5x + 6\) is explicit as it expresses \(y\) in terms of \(x\).

However, functions that are expressed implicitly, such as \(xy = 9\) has the two variables on the same side. In this case, \(xy = 9\) can be arranged to \(y = \frac{9}{x}\), but this isn't as straightforward for all implicit functions.


Example:

• Find derivative for the function \(x^2 + y^2 = 16\).

First we differentiate both sides with respect to \(x\): \[\frac{d(x^2 + y^2)}{dx} = \frac{d(16)}{dx}\]

Next we break up the left hand side into seperate fractions: \[\frac{d(x^2)}{dx} + \frac{d(y^2)}{dx} = 0\]

We know that the derivative of \(x^2\) is \(2x\). However, it is harder to differentiate \(y^2\) with respect to \(x\), so we apply the chain rule: \[\frac{d(y^2)}{dx} = \frac{d(y^2)}{dy} * \frac{dy}{dx}\]

We can now differentiate \(y^2\) with respect to \(y\), which is \(2y\), but we don't know what \(\frac{dy}{dx}\) is. So, we have the following: \[\eqalign{\frac{d(y^2)}{dx} &= \frac{d(y^2)}{dy} * \frac{dy}{dx} \\ &= 2y\frac{dy}{dx}}\]

Finally, we substitute this back into the expression for the implicit derivative: \[\frac{d(x^2)}{dx} + \frac{d(y^2)}{dx} = 0\] \[2x + 2y\frac{dy}{dx} = 0\] \[2y\frac{dy}{dx} = -2x\] \[\eqalign{\frac{dy}{dx} &= \frac{-2x}{2y} \\ &= \frac{-x}{y}}\]

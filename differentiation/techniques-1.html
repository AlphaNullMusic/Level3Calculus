Differentiation

- Differentiating from First Principles

We already know some of the rules of differentiating from calculus last year. For example: \(x^2\) becomes \(2x\) and \(5x\) becomes \(5\). But, how can we figure this out from first principles?


Differentiation is about finding the rate of change at a single point of a function. It is hard to find the gradient from a graph as it is changing at every point of the curve. However, we can find two points on a graph and move them very close to each other so that the secant (line between the two points) gets smaller.


We can name these points \(x\) and \(x + h\), where \(h\) is the difference between the two points on the x-axis. As \(h\) approaches zero, the average rate of change will get closer to the gradient of the tangent at the point. This means that we can use limits to solve it.

Therefore: \[f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}\] Where \(f'(x)\) is the rate of change of \(f(x)\) (the gradient function).


Example:

• Differentiate \(x^2\) from first principles.

Firstly, we identify \(f(x+h)\): \[f(x) = x^2\] \[f(x+h) = (x+h)^2\]

Then we write the derivative from first principles: \[\eqalign{f'(x) &= \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} \\ &= \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h}}\]

We can expand and cancel terms to simplify the fraction: \[\eqalign{\lim_{h \to 0} \frac{(x^2 + 2xh + h^2) - x^2}{h} &= \lim_{h \to 0} \frac{2xh + h^2}{h} \\ &= \lim_{h \to 0} 2x + h}\]

Finally, we take the limit. As \(h\) approaches zero, it effectively cancels out (because \(2x + 0.0000000001 \approx 2x\)). Therefore, we know that the derivative of \(x^2\) is \(2x\). We can apply this to other functions to generate rules for differentiation.

- Differentiating Polynomials

The rule for differentiating polynomials is as follows:

If \(f(x) = ax^n\) then \(f'(x) = anx^{n-1}\).

In basic terms, multiply the coefficient of \(x\) by \(n\) and subtract 1 from the power.

Note: \(f(x) = ax^n\) is in index form. You may need to manipulate the function to use it in index form.

There are six situations that you need to know:

  1. If the function is already in index form, then use the rule directly: \[f(x) = 2x^5 + 3x^2 + 8x - 2\] \[f'(x) = 10x^4 + 6x + 8\]
  2. If the function has fractions and x is on the numerator, seperate the fraction and use the rule: \[\eqalign{f(x) &= \frac{3x^2}{4} + 2x \\ &= \frac{3}{4}x^2 + 2x}\] \[f'(x) = \frac{3}{2}x + 2\]
  3. If the function has fractions and x is on the denominator, rewrite the x term with a negative index, use the rule, then convert the answer back to the form the question was in: \[\eqalign{f(x) &= \frac{5}{4x^3} + \frac{x}{2} \\ &= \frac{5}{4}x^{-3} + \frac{1}{2}x}\] \[\eqalign{f'(x) &= -\frac{15}{4}x^{-4} + \frac{1}{2} \\ &= -\frac{15}{4x^4} + \frac{1}{2}}\]
  4. If the function is a product of factors, expand out the brackets: \[\eqalign{f(x) &= (x+2)(x-4) \\ &= x^2 - 2x - 8}\] \[f'(x) = 2x - 2\]
  5. If the function is expressed as a complete fraction, with x on both sides, break the function up, simplify each one and use the rule: \[\eqalign{f(x) &= \frac{5x^4 + 3x^2 + 2x - 5}{2x} \\ &= \frac{5x^4}{2x} + \frac{3x^2}{2x} + \frac{2x}{2x} - \frac{5}{2x} \\ &= \frac{5x^3}{2} + \frac{3x}{2} + 1 - \frac{5x^{-1}}{2}}\] \[\eqalign{f'(x) &= \frac{15x^2}{2} + \frac{3}{2} + \frac{5x^{-2}}{2} \\ &= \frac{15x^2}{2} + \frac{3}{2} + \frac{5}{2x^2} \\ &= \frac{15x^4 + 3x^2 + 5}{2x^2}}\]
  6. If the fraction has surds, convert them to indicies (remembering that \(\sqrt[n]{x^m} = x^{\frac{m}{n}}\)), and use the rule: \[\eqalign{f(x) &= \sqrt[3]{x} + \sqrt[4]{x^2} \\ &= x^{\frac{1}{3}} + x^{\frac{2}{4}}}\] \[\eqalign{f'(x) &= \frac{1}{3}x^{-\frac{2}{3}} + \frac{1}{2}x^{-\frac{1}{2}}\\ &= \frac{1}{3\sqrt[3]{x^2}} + \frac{1}{2\sqrt{x}}}\]

- Differentiating Composite Functions (The Chain Rule)

A composite function is where one function is substituted for a variable in another. For example: if we had \(f(x) = x^5\) and \(g(x) = x^3 - 2x + 1\), then \(f(g(x))\) is the composite function, where \(g(x)\) was substituted into \(f(x)\).

The composite function for this example is then: \[\eqalign{f(g(x)) &= f(x^3 -2x + 1) \\ &= (x^3 - 2x + 1)^5}\]


In order to differentiate composite functions, we need to use the chain rule, which states that if \(f(x) = g(h(x))\) then \(f'(x) = g'(h(x)) * h'(x)\).

This can be rewritten as: if \(f(x) = v(u)\) then \(f'(x) = v' * u'\).

Example:

• Differentiate \(y = (3x^2 + 2x - 1)^3\).

First we start with the outer function (\(y=u^3\)) and then multiply the answer by the inner function (\(u=3x^2 + 2x - 1\)) differentiated: \[\eqalign{\frac{dy}{dx} &= \frac{dy}{du} * \frac{du}{dx} \\ &= 3u^2 * (6x+2)}\]

Finally, we substitute \(u\) back in to give the answer: \[\frac{dy}{dx} = 3(6x+2)(3x^2 + 2x - 1)^2\]

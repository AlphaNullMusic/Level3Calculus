Differentiation

- Differentiating Products of Two or More Functions

Some functions are products of two seperate functions, such as: \(f(x) = g(x) * h(x)\).

Previously, we differentiated an example in this form: \(f(x) = (x+2)(x-4)\). We did this by expanding out the brackets to put it in index form and then using the polynomial rule.


However, some functions may be too hard to expand, such as: \(2\sqrt[3]{x^5}(x^3 + 5x - 6)\).

Rather than expanding these into index form, we can use the product rule. The product rule states that for any function expressed as \(f(x) = g(x) * h(x)\), then \(f'(x) = g'(x) * h(x) = g(x) * h'(x)\).

Note: you may be required to prove the product rule by use of first principles for excellence.

Example:

• Differentiate \(y=(3x^4 + 2x + 3)(5x^5 - 4x + 6)\).

First, we let \(u=3x^4 + 2x + 3\) and \(v=5x^5 - 4x + 6\).

Then we find the derivatives of \(u\) and \(v\): \[u' = 12x^3 + 2\] \[v' = 25x^4 - 4\]

Then we use the product rule: \[\eqalign{y' &= u'v + uv' \\ &= (12x^3 + 2)(5x^5 - 4x + 6) + (3x^4 + 2x + 3)(25x^4 - 4)}\]

- Differentiating Quotients

Some functions are quotients of two seperate functions (one function divided by another), for example: \[\eqalign{f(x) &= \frac{g(x)}{h(x)} \\ &= \frac{2\sqrt[3]{x^5}}{x^3 + 5x - 6}}\]

To differentiate this function, we can use the quotient rule, which states that if: \[f(x) = \frac{g(x)}{h(x)}\]

then: \[f'(x) = \frac{g'(x) * h(x) - g(x) * h'(x)}{(h(x))^2}\]


Example:

• Differentiate: \[f(x)=\frac{2x^2 + x + 1}{\sqrt[4]{x}}\]

First we let \(u = 2x^2 + x + 1\) and \(v = \sqrt[4]{x}\).

Then we find the derivatives of \(u\) and \(v\): \[u' = 4x + 1\] \[v' = \frac{1}{4\sqrt[4]{x^3}}\]

Finally we substitute into the quotient rule to get: \[f'(x) = \frac{(4x + 1)\sqrt[4]{x} - (2x^2 + x + 1)(\frac{1}{4\sqrt[4]{}x^3})}{\sqrt{x}}\]

- Differentiating Exponential Functions

The exponential function \(f(x) = e^x\) is a special in that it is the only function to equal its derivative at all points. This is because the gradient of the tangent at any point is equal to the y co-ordinate at that point. Therefore, if \(f(x) = e^x\) then \(f'(x) = e^x\).

Sometimes we may have to differentiate composite exponential function, such as \(y = 5e^{3x-4}\). In this case, we use the chain rule which can be shortened to a simple rule: if \(f(x) = ae^{g(x)}\) then \(f'(x) = a * g'(x) * e^{g(x)}\).

Example:

• Differentiate \(f(x) = -2e^{(-5x^2 + 3x - 5)}\).

First we let \(g(x) = -5x^2 + 3x - 5\) so \(f(x) = -2e^{g(x)}\).

We know that \(g'(x) = -10x + 3\), so the derivative of \(f(x)\) is: \[\eqalign{f'(x) &= -2 * g'(x)e^{g(x)} \\ &= 2(10x - 3)e^{(-5x^2 + 3x - 5)}}\]

- Differentiating the Natural Logarithm Function

The exponential and natural logarithm functions are related and inverse to each other. This means that if \(y = e^x\) then \(\log_{e} y = x\).

Note: \(log_{e} x = y\) is usually expressed as \(\ln{x}\).

The rule for differentiating the natural logarithm is if \(y = \ln{x}\) then \(\frac{dy}{dx} = \frac{1}{x}\).


Proof:

If we let \(log_{e} x = y\) then we can find that \(x = e^y\). Differentiating that gives itself: \[\frac{dx}{dy} = e^y\]

This is inverse to what we usually work with (differentiating the function of \(y\) with respect to \(x\)), so we flip it: \[\frac{dy}{dx} = \frac{1}{e^y}\]

Finally, we substitute \(x = e^y\) back into the equation: \[\frac{dy}{dx} = \frac{1}{x}\]


Combining this with the chain rule will give us a rule to differentiate composite functions with natural logarithms:

If \(f(x) = \ln{g(x)}\) then: \[\eqalign{f'(x) &= g'(x) * \frac{1}{g(x)} \\ &= \frac{g'(x)}{g(x)}}\]


Example:

• Differntiate \(y = \ln{((3x+7)^3)}\)

Using the chain rule to differentiate \(g(x)\) we get: \[\eqalign{g'(x) &= 3(3x + 7)^2 * 3 \\ &= 9(3x + 7)^2}\]

Then we substitute it into the rule to get the answer: \[\eqalign{\frac{dy}{dx} &= \frac{g'(x)}{g(x)} \\ &= \frac{9(3x + 7)^2}{(3x + 7)^3} \\ &= \frac{9}{3x + 7}}\]

- Differentiating Trigonometric Functions

All basic trigonometric functions can be differentiated. You will be given a table with the original and derivative functions for your exam. They can also be worked out using first principles and trigonometric ratios.


Function (\(f(x)\)) Derivative (\(f'(x)\))
\(\sin{x}\) \(\cos{x}\)
\(\cos{x}\) \(-\sin{x}\)
\(\tan{x}\) \(\sec^2{x}\)
\(\csc{x}\) \(-\csc{x} \cot{x}\)
\(\sec{x}\) \(\sec{x} \tan{x}\)
\(\cot{x}\) \(-\csc^2{x}\)

Note:

- \(\csc{x}\) can also be expressed as \(\text{cosec } x\).

- trigonometric functions are often written using this notation: \((\sin{x})^2\) is written as \(\sin^2{x}\).

- use the chain rule for composite functions (e.g: \(f(x) = \sin{(g(x))}\).

Example:

• Differentiate \(y = \sin{x} \cos{x}\).

Let \(u = \sin{x}\) and \(v = \cos{x}\).

We know from the table that \(u' = \cos{x}\) and \(v' = -\sin{x}\).

Therefore, we can use the product rule to give the answer: \[\eqalign{y' &= u'v + uv' \\ &= \cos{x}\cos{x} + \sin{x}(-\sin{x}) \\ &\ (\cos{x})^2 - (\sin{x})^2 \\ &= \cos^2{x} - \sin^2{x}}\]
